Um Web Player de alto desempenho com estética Retro-Futurista (Glassmorphism), focado em streams de alta qualidade (HLS) e interface otimizada para dispositivos móveis.

## ✨ Funcionalidades
- 🖥️ **Interface Glassmorphism**: Visual moderno com transparências e desfoque.
- 📱 **Totalmente Responsivo**: Ajuste automático para smartphones e desktops.
- 📡 **Protocolo HLS**: Suporte nativo para streams `.m3u8` via Hls.js.
- 🎞️ **Efeito Scanlines**: Simulação de tela CRT sutil para imersão total.
